# Alaska AI - Python Version

تطبيق ألاسكا مبني باستخدام Python (Flask) وواجهة Bootstrap.

### المتطلبات
* Python 3.8+
* مفتاح Google Gemini API

### طريقة التثبيت والتشغيل

1. **تثبيت المكتبات:**
   افتح موجه الأوامر (Terminal) واكتب:
   ```bash
   pip install -r requirements.txt
   ```

2. **إعداد المفتاح:**
   * قم بإنشاء ملف جديد باسم `.env` بجانب ملف `app.py`.
   * افتحه واكتب بداخله:
     ```
     API_KEY=AIzaSy... (ضع مفتاحك هنا)
     ```

3. **تشغيل التطبيق:**
   اكتب الأمر:
   ```bash
   python app.py
   ```

4. **الاستخدام:**
   افتح المتصفح على الرابط: `http://127.0.0.1:5000`
